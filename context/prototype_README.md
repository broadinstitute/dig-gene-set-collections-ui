# CFDE-REVEAL Prototype

Local prototype for provenance-aware dataset recommendation built under `ui_test/prototype/`.

## Current scope

- Sources: GTEx, MoTrPAC, GEO, 4DN, HuBMAP, TCGA SpliceSeq, DepMap PRISM, TCGA/GDC, JUMP Cell Painting, and Cal-Repository
- Current corpus: 48 cards
- Retrieval evidence: overlap, enrichment-style, latent signature similarity
- UI: Streamlit

## Build artifacts

Rebuild the current corpus and retrieval backend:

```bash
../.venv/bin/python ui_test/prototype/scripts/build_demo_inputs.py
../.venv/bin/python ui_test/prototype/scripts/run_extractions.py
../.venv/bin/python ui_test/prototype/scripts/build_additional_inputs.py
../.venv/bin/python ui_test/prototype/scripts/run_additional_extractions.py
../.venv/bin/python ui_test/prototype/scripts/build_manifest.py
../.venv/bin/python ui_test/prototype/scripts/build_retrieval_index.py
```

Or run the thin-slice extraction bundle:

```bash
bash ui_test/prototype/scripts/run_thin_slice.sh
../.venv/bin/python ui_test/prototype/scripts/build_retrieval_index.py
```

## Run the app

Create the local UI venv once:

```bash
python3 -m venv ui_test/.venv
ui_test/.venv/bin/pip install -r ui_test/prototype/app/requirements.txt
```

If the venv already exists, refresh dependencies with:

```bash
ui_test/.venv/bin/pip install -r ui_test/prototype/app/requirements.txt
```

Start Streamlit:

```bash
ui_test/.venv/bin/python -m streamlit run ui_test/prototype/app/app.py --server.headless true --server.port 8765
```

## Notes

- GTEx signatures are built from public GTEx API expression summaries.
- A second GTEx track now adds grouped tissue-versus-rest bulk RNA cards built from five local open-access GTEx tissue splits.
- MoTrPAC transcriptomics signatures are mapped from native rat identifiers into a human comparison space before extraction.
- Additional MoTrPAC WAT proteomics and phosphoproteomics runs are staged under `ui_test/prototype_inputs/` and emitted under `ui_test/prototype_outputs/`.
- The first additional 4DN run uses released processed ATAC peaks from `4DNFIUALWN8X`.
- A second 4DN run now uses released processed H3K27ac ChIP-seq peaks from `4DNFIGINV1VI`, converted from `bigBed` to plain BED for `chipseq_peak`.
- The current HuBMAP run uses the published liver processed RNA-seq data product `6327fd11-974c-48d1-910d-9b981a84c28e`, exporting the `unscaled` layer plus published `leiden` groups for `sc_rna_marker`.
- The current BRCA splicing card uses the prepared public TCGA SpliceSeq `subset5000_norm` matrix plus the local BRCA splicing bundle.
- The current PRISM cards use the staged quick public subset grouped by primary tissue.
- The current CNV cards use a local 50-sample subset of public TCGA-BRCA segments.
- The current morphology cards use the public JUMP-Target U2OS held-out query panel against the staged `morphology_jump_target_pilot_U2OS_48_v1` bundle.
- The current calorimetry cards use the public Cal-Repository Figure 2 UC Davis study in both ontology-mapper and public-bundle profile-query modes.
- The current methylation card uses the public GSE42752 tumor-versus-matched-normal staged CpG differential table.
- SPARC, Bridge2AI/CM4AI, and SenNet remain unimplemented in this prototype pass because a clean direct-download processed matrix path was not confirmed from their public portals.
- GEO uses the local Kang IFN-beta long-format DE table already present in this workspace.
- The current UI expects graph-ready provenance JSON under `ui_test/prototype/provenance_graph/`, generated by `build_manifest.py`.
